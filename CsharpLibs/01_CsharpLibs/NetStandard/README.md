# CsharpLibs

Conjunto de librerias escritas en .Net Standard.

Las librerias escritas en .Net Standard son librerias multiplataforma, osea que son compatibles con .Net Framework, .Net CORE, Xamarin, etc.

Estas son las librerias actuales:

- **Databases:** Libreria para la gestion de bases de datos como SQLite

- **DirectorioArchivos:** Libreria para facilitar la gestion de archivos y directorios como copiar, ordenar rutas etc.

- **Logs:** Libreria para crear un fichero de escritura de logs

- **ModulosCifrado:** Libreria para usar algoritmos de cifrado mas sencillamente como AES etc

- **Respuestas:** Libreria que contiene un objeto para gestionar resultados y excepciones principalemente en aplicaciones Web, hace que se devuelva los resultados en un formato parecido o igual
