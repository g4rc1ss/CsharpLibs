# CsharpLibs

Conjunto de librerías escritas en .Net Standard.

Las librerías escritas en .Net Standard son librerías multiplataforma, osea que son compatibles con .Net Framework, .Net CORE, Xamarin, etc.

Estas son las librerías actuales:

- **Databases:** Librería para la gestión de bases de datos como SQLite

- **DirectorioArchivos:** Librería para facilitar la gestión de archivos y directorios como copiar, ordenar rutas etc.

- **Logs:** Librería para crear un fichero de escritura de logs

- **ModulosCifrado:** Librería para usar algoritmos de cifrado mas sencillamente como AES etc

- **Respuestas:** Librería que contiene un objeto para gestionar resultados y excepciones principalmente en aplicaciones Web, hace que se devuelva los resultados en un formato parecido o igual
